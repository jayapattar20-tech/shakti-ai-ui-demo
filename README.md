# SHAKTI AI — UI/UX Prototype

A front-end concept demo showcasing the planned features and user experience for SHAKTI, an AI-powered women's safety application. This prototype demonstrates the interface design, user flows, and feature concepts (SOS system, risk mapping, threat detection) — the ML models and backend integrations described are part of the planned architecture and not yet implemented in this version.

Built with AI-assisted development (HTML/CSS/JS), with design and feature direction by me.

## Features Demonstrated
- Multi-trigger SOS countdown system (voice, scream, shake, panic button)
- AI risk mapping with safe/danger zone visualization
- Simulated threat detection and emergency alert flow

## Status
This is a prototype/concept stage — UI and interactions are functional, but backend services and ML models are planned, not yet built.
